# Practice: Writing Unit Tests

## Assignment
This project implements and tests a basic shopping cart module.

## Required Methods

1. addItem(cart, item, quantity)
2. removeItem(cart, item)
3. getTotalItems(cart)

## Files

- cart.js
- tests/cart.test.js
- package.json
- package-lock.json
- .gitignore
- INSTRUCTIONS_AND_PROCESS.md
- TESTING_NOTES.md
- WORK_SHOWN.md

## How to Run

npm install

npm test

## Clone Instructions

Example repository workflow:

git clone https://github.com/chrismusic1325/-practice-writing-unitTests--Christopher---H-.git

cd shopping-cart

npm install

npm test

git add .

git commit -m "Update project"

git push
