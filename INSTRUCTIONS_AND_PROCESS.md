# Instructions and Process

## Step 1: Create the Project
I created a shopping-cart project folder.

## Step 2: Initialize Node
I used npm init -y to create package.json.

## Step 3: Install Jest
I installed Jest using npm install --save-dev jest.

## Step 4: Add Test Script
I changed package.json so npm test runs jest.

## Step 5: Create cart.js
I created the shopping cart module with addItem, removeItem, and getTotalItems.

## Step 6: Export Functions
I used module.exports so the test file can import the functions.

## Step 7: Create Test Folder
I created a tests folder.

## Step 8: Create cart.test.js
I wrote unit tests for positive tests, negative tests, and edge cases.

## Step 9: Run Tests
I used npm test to check that the tests pass.

## Step 10: Refactor
If tests failed, the plan would be to update cart.js and run npm test again.

## Step 11: Push to GitHub
I committed the project and pushed it to GitHub.
